二、使用的基础配置

安装的过程中功能都已预先配置完成，更多的配置可在：

  • .env文件实现大部分配置。
  • (Laravel)发布的配置文件或(Lumen)bootstrap/app.php中配置
  • AppserviceProvider的boot方法

env中的配置

标准树(API_STANDRDS_TRUE)

标准树有三个不同的值

  • x:未注册的，树用于本地或私有环境
  • prs:个人树，用于非商业销售的项目
  • vnd:供应商树，主要用于公开的及商业销售的项目

子类型(API_SUBTYPE)

子类型通常是应用或项目的简称，

API_PREFIX=api

API_DOMAIN=api.myapp.com

API_VERSION=V

API_NAME=My API

API_CONDITIONAL_REQUEST=false

API_STRICT=false

API_DEBUG=true
`
